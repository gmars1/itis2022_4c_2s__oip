## before all

`pip install -r requirements.txt`

## task1

> starting from project root

`python -m task1.target_list_generator` - заполнить `task1/target_list.txt`

`python -m task1.crawler` - скачать страницы в `task1/crawled/`; заполнить `task1/index.txt`

## task2 

> starting from project root

`python -m task2.setup` - загрузить необходимые библиотеки

`python -m task2.main` - заполнить `task2/lemmas.txt`, `task2/tokens.txt`; требуется `/task1/crawled/`
